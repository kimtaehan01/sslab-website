# 디자인 개선 사항

## 1. Home 페이지 개선
- 히어로 섹션에 타이핑 애니메이션 또는 카운트업 효과 추가
- 통계 바에 카운트업 애니메이션 추가
- About 섹션에 about-pattern 배경 이미지 활용

## 2. Layout 개선
- 모바일 네비게이션 개선 - 풀스크린 오버레이 스타일
- 네비게이션 스크롤 시 배경 변화 효과

## 3. Research 페이지 개선
- 시스템 보안과 임베디드 시스템에 이미지 없음 - about-pattern 활용

## 4. Members 페이지 개선
- 대학원생 3명 추가 (현재 데이터 있음)
- 카드 디자인 더 세련되게

## 5. Contact 페이지
- 구글맵 embed URL 개선 (현재 실제 한남대 위치 표시됨)
