## 아이디어 1: "Terminal Noir" — 해커/터미널 미학

<text>
### Design Movement
**Cyberpunk Terminal Aesthetic** — 터미널 인터페이스와 코드 에디터에서 영감을 받은 다크 모드 중심의 디자인. 시스템 소프트웨어 랩의 정체성을 시각적으로 직접 표현한다.

### Core Principles
1. **코드가 곧 디자인이다**: 모노스페이스 타이포그래피와 터미널 프롬프트 스타일의 UI 요소
2. **어둠 속의 빛**: 깊은 다크 배경 위에 네온 그린/시안 액센트로 시선 유도
3. **기능적 미니멀리즘**: 불필요한 장식 없이 정보 밀도를 높이는 레이아웃
4. **시스템 레벨 신뢰감**: 기술적 깊이를 시각적으로 전달

### Color Philosophy
- 배경: `#0a0a0f` (깊은 미드나잇 블랙)
- 텍스트: `#e0e0e0` (소프트 화이트)
- 주 액센트: `#00ff88` (터미널 그린)
- 보조 액센트: `#00d4ff` (시안 블루)
- 코드 하이라이트: `#ff6b6b` (에러 레드), `#ffd93d` (워닝 옐로)

### Layout Paradigm
- 좌측 고정 사이드바 네비게이션 (터미널 파일 트리 스타일)
- 메인 콘텐츠는 코드 에디터의 탭 구조를 차용
- ASCII 아트 디바이더와 커서 블링크 애니메이션

### Signature Elements
1. 타이핑 애니메이션으로 텍스트가 나타나는 히어로 섹션
2. `$ sslab --help` 스타일의 명령어 프롬프트 네비게이션
3. 매트릭스 스타일의 배경 파티클

### Interaction Philosophy
- 호버 시 터미널 커서 스타일 변경
- 클릭 시 `> executing...` 피드백
- 스크롤 시 코드가 컴파일되는 듯한 트랜지션

### Animation
- 텍스트 타이핑 효과 (typewriter)
- 글리치 효과 트랜지션
- 스캔라인 오버레이

### Typography System
- 제목: `JetBrains Mono` Bold
- 본문: `JetBrains Mono` Regular
- 액센트: `Fira Code` with ligatures
</text>

## 아이디어 2: "Geometric Precision" — 스위스 모더니즘 + 테크 퓨전

<text>
### Design Movement
**Neo-Swiss Modernism meets Tech** — 스위스 국제 타이포그래피 스타일의 정밀함에 현대 테크 기업의 세련됨을 결합. 그리드 시스템의 엄격함과 대담한 타이포그래피로 학술적 권위와 기술적 혁신을 동시에 전달한다.

### Core Principles
1. **수학적 정밀함**: 엄격한 8pt 그리드 시스템과 황금비 기반 레이아웃
2. **대담한 대비**: 극단적인 폰트 사이즈 대비와 흑백 + 단일 강조색
3. **구조적 명확성**: 정보 계층이 시각적으로 즉시 파악되는 레이아웃
4. **공백의 힘**: 여백을 적극적 디자인 요소로 활용

### Color Philosophy
- 배경: `#fafafa` (오프 화이트)
- 텍스트: `#1a1a2e` (딥 네이비)
- 주 액센트: `#e63946` (시그널 레드) — 주의를 끄는 단일 강조색
- 보조: `#457b9d` (스틸 블루) — 링크와 인터랙티브 요소
- 서피스: `#f1faee` (민트 화이트)

### Layout Paradigm
- 비대칭 2단 레이아웃: 좌측 60% 콘텐츠 + 우측 40% 보조 정보
- 섹션 간 대각선 디바이더로 시각적 흐름 생성
- 오버사이즈 타이포그래피가 레이아웃 그리드를 의도적으로 깨뜨림

### Signature Elements
1. 초대형 숫자/텍스트가 배경에 워터마크처럼 배치
2. 기하학적 도형(원, 삼각형, 선)이 콘텐츠와 오버랩
3. 흑백 사진에 단색 오버레이 처리

### Interaction Philosophy
- 호버 시 기하학적 도형이 회전하거나 변형
- 정확한 타이밍의 마이크로 인터랙션 (200ms ease-out)
- 스크롤 시 텍스트와 도형의 패럴랙스 효과

### Animation
- 요소 진입 시 슬라이드 + 페이드 (좌→우 방향성)
- 숫자 카운트업 애니메이션
- 기하학적 도형의 미세한 회전 루프

### Typography System
- 제목: `Space Grotesk` Bold/Black (72-120px 히어로, 36-48px 섹션)
- 본문: `IBM Plex Sans` Regular (16-18px)
- 강조: `Space Grotesk` Medium + 레드 액센트
- 캡션: `IBM Plex Mono` Light (12-14px)
</text>

## 아이디어 3: "Digital Fabric" — 유기적 테크 미니멀리즘

<text>
### Design Movement
**Organic Tech Minimalism** — 차가운 기술과 따뜻한 유기적 형태의 조화. 소프트한 그라데이션, 유기적 곡선, 그리고 넉넉한 여백으로 접근성과 세련됨을 동시에 추구한다. Apple의 Human Interface와 일본 미니멀리즘의 교차점.

### Core Principles
1. **부드러운 기술**: 딱딱한 직선 대신 유기적 곡선과 소프트 쉐도우
2. **호흡하는 레이아웃**: 콘텐츠 사이에 충분한 여백으로 시각적 편안함
3. **점진적 노출**: 스크롤에 따라 정보가 자연스럽게 드러나는 구조
4. **촉각적 깊이감**: 글래스모피즘과 미세한 그라데이션으로 레이어 표현

### Color Philosophy
- 배경: `#f8f9fc` → `#eef1f8` (미세한 블루 그라데이션)
- 텍스트: `#2d3142` (웜 다크 그레이)
- 주 액센트: `#4361ee` (일렉트릭 블루)
- 보조: `#7209b7` (딥 퍼플)
- 서피스: `rgba(255,255,255,0.7)` (프로스트 글래스)
- 소프트 그라데이션: `#4361ee` → `#7209b7` (블루 투 퍼플)

### Layout Paradigm
- 풀 와이드 섹션과 컨테이너 섹션의 교차 리듬
- 카드 기반 콘텐츠 블록이 유기적으로 배치
- 블롭(blob) 형태의 배경 요소가 섹션 경계를 부드럽게 연결

### Signature Elements
1. 글래스모피즘 카드 (backdrop-blur + 반투명 배경)
2. 유기적 블롭 그라데이션 배경 요소
3. 미세한 그레인 텍스처 오버레이

### Interaction Philosophy
- 호버 시 카드가 미세하게 떠오르는 3D 효과
- 부드러운 스프링 물리 기반 애니메이션
- 마우스 위치에 반응하는 배경 그라데이션

### Animation
- Framer Motion 기반 스크롤 트리거 애니메이션
- 스프링 물리 (stiffness: 100, damping: 15)
- 블롭 배경의 느린 모핑 애니메이션

### Typography System
- 제목: `Outfit` SemiBold/Bold
- 본문: `Plus Jakarta Sans` Regular/Medium
- 코드/기술: `Fira Code` Regular
- 캡션: `Plus Jakarta Sans` Light
</text>

## 선택: 아이디어 2 — "Geometric Precision"

시스템 소프트웨어 연구실의 정체성에 가장 부합하는 디자인. 스위스 모더니즘의 정밀함은 시스템 소프트웨어의 엄밀함과 일맥상통하며, 대담한 타이포그래피와 기하학적 요소는 컴퓨터공학과의 기술적 전문성을 시각적으로 전달한다. 학술적 권위와 현대적 세련됨의 균형이 한남대학교 연구실 웹사이트로서 적합하다.
