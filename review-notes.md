# 디자인 리뷰 결과

## 홈페이지
- 히어로 섹션: 배경 이미지 + 그라데이션 오버레이 잘 적용됨
- 통계 바: 스크롤 시 보이지 않음 - 확인 필요 (카운트업 동작 확인 필요)
- About 섹션: 비대칭 레이아웃 + 랩실 이미지 잘 표시
- 연구 분야 카드: 다크 배경에 4개 카드 2x2 그리드, 아이콘 + 번호 잘 표시
- Featured 이미지 섹션: 연구 시스템 이미지 + 접근법 설명 잘 표시
- CTA 섹션: 다크 배경 + 도트 패턴, 버튼 2개 잘 표시
- 푸터: 4컬럼 레이아웃, 기하학적 디바이더 잘 표시
